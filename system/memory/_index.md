# Knowledge Graph — Map of Content

> Entry point to all topic files. Follow [[links]] to load only what's relevant.

## How to Use
- Each `[[topic]]` maps to `memory/topics/topic.md`
- Only load topics relevant to the current conversation
- Create new topic files freely — one subject per file
- Add [[links]] between related topics
- Run `/graduate` periodically to promote daily note insights here

## Topics

_Add your first topics here as you work. Format:_

```
## [Category]
- [[topic-name]] — one-line description
```

## Example Structure

```
## Projects
- [[project-x]] — Main product, current sprint goals
- [[project-y]] — Side project, exploring new tech

## People
- [[teammate-name]] — Role, preferences, working style

## Decisions
- [[architecture-choice]] — Why we chose X over Y

## Learnings
- [[domain-insight]] — Key insight from research
```
