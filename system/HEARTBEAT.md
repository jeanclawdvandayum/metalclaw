# HEARTBEAT.md — Periodic Check-In

## What to Do Each Heartbeat

Pick 1-2 from this list (rotate through them, don't do all every time):

- [ ] Check emails — anything urgent or actionable?
- [ ] Check calendar — events in next 24h?
- [ ] Check project status — git repos, open PRs, stale branches?
- [ ] Memory maintenance — any recent daily notes worth graduating to topic files?
- [ ] Check HOLDS.md — any expired holds to drop?

## Rules
- Keep it lightweight — heartbeats should be quick
- If nothing needs attention, reply HEARTBEAT_OK
- Don't trigger expensive sub-agent chains from heartbeats
- Late night (23:00-08:00): HEARTBEAT_OK unless something is genuinely urgent
